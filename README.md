# Big Backs Assemble — React Prototype

A front-end prototype for the Big Backs Assemble social restaurant discovery app.

## Setup

Make sure you have **Node.js 18+** installed.

```bash
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev
```

The app will open at **http://localhost:5175**

## Pages & Routes

| Route | Page |
|---|---|
| `/` | Landing page |
| `/login` | Log in |
| `/signup` | Sign up |
| `/feed` | Home feed |
| `/explore` | Explore restaurants |
| `/restaurant/:id` | Restaurant detail |
| `/profile` | User profile |
| `/wishlist` | My wishlist |
| `/messages` | Messages & chat |
| `/add-review` | Add a review |

## Notes

- All data is hard-coded
- No backend — authentication is simulated (any input works)
